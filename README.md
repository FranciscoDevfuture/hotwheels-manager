# 🚗 Hot Wheels Manager

Sistema desktop desenvolvido em Python para gerenciamento de carrinhos colecionáveis Hot Wheels.

O projeto permite cadastrar, visualizar e organizar uma coleção de forma simples, intuitiva e funcional, simulando um software real de controle de inventário.

---

## 📌 Funcionalidades

✅ Cadastro de carrinhos  
✅ Nome do carrinho  
✅ Modelo  
✅ Cor  
✅ Ano  
✅ Listagem da coleção  
✅ Interface gráfica com Tkinter  
✅ Estrutura orientada a objetos (Classe Carrinho)

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Tkinter
- Programação Orientada a Objetos (POO)
- Git e GitHub
